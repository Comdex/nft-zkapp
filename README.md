# nft-zkapp: a simple privacy NFT based on mina zkApp

NFT Proof of Concept based on mina actions/reducer scheme.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
